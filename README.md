Problem statement:
The goal of this project is to analyze candidate placement data and build a Job Acceptance Prediction System,
predict whether a candidate will accept or reject a job offer, identify key factors influencing job acceptance decisions.

Proposed solution:
2 ML models:
logistic regression
random forest classifier

Both trained and tested with accuracy of logistic regression = 85% and random forest classifier = 90%

app:
streamlit interactive dashboard developed using rf_model
